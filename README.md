## 命名规范

- 文件名称

  小写英文单词，单词用连字符分隔，同一模块的 js 和 样式文件名字要统一

## 项目规范

采用 ES6 语法编码

缩进 **2** 个空格

代码提交写明注释(动词开头＋描述开发的功能或解决的问题)

### 开始开发

- 获取代码

  ```bash
    git clone https://github.com/fe-charts/fe-charts.git
  ```

- 安装依赖

  ```bash
    yarn # strongly recommend
  ```

- 开发

  ```bash
    yarn start # strongly recommend
  ```

- 测试

  ```bash
    yarn test
  ```

- 构建

  ```bash
    yarn build
  ```
